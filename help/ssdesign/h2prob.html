H2-optimization problem

The goal of the optimization is to minimize the the H2-norm of the sampled-data system [1], which is similar to the H2-norm of LTI systems. In stochastic setting, this equals the average output variance under the input described as a centered white noise with unit spectral density [2].

State-space algorithms for solving this problems were developed in [1] and [3]-[5] using various approaches. Unfortunately, they can hardly cope with time-delayed systems.

The idea bases on construction of an equivalent discrete-time model of the problem. The function sdh2reg returns the optimal digital controller and the minimal attainable value of the cost function: 

   F = tf(1, [1 1 0]);
   sys = [F F;-F -F];
   T = 0.1;
   [K,cost] = sdh2reg ( sys, T );
   K = zpk(K)
   cost
