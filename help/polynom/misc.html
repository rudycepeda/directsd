Miscellaneous functions

Reciprocal polynomial
Reciprocal polynomial (the one with the inverve order of coefficients) can be found using the function recip: 
   a = poln( [1 2 3 4], 'z' )

a: polynomial
  z^3 + 2 z^2 + 3 z + 4

   b = recip ( a )

b: polynomial
  4 z^3 + 3 z^2 + 2 z + 1
Removing roots at the origin
Function delzero removes all zeros of a polynomial at the origin and returns the reduced polynomial and the number of removed zeros: 
   a = poln( [1 2 0 0], 'z' )

a: polynomial
  z^3 + 2 z^2

   [b,nz] = delzero ( a )

b: polynomial
  z + 2
nz =
     2
Transformation to ZPK-model
To transform a polynomial or a pair of polynomials to the corresponsing zero-pole-gain model of Control System Toolbox the overloaded zpk function can be used. The function can be called with one argument 
   n = poln( [2 3], 'z' )

n: polynomial
  2 z + 3

   f = zpk ( n )

f: zero-pole-gain model 1 x 1

      2 (z+1.5)

Sampling time: 1
or with two arguments (when a polynomial ratio is transformed into a zpk-model) 
   d = poln( [1 2 0], 'z' )

d: polynomial
  z^2 + 2 z

   f = zpk ( n, d )

f: zero-pole-gain model 1 x 1

      2 (z+1.5)
      ---------
       z (z+2)

Sampling time: 1
Discretization
Discretization of a polynomial means transformation to a discretized form. In this case roots of the polynomial are transformed to their discretized version so that a root ai is changed for zi=exp(-ai*T) (function c2d) or zi=exp(ai*T) (function c2z). For example: 
   a = poln([-1 -2 -3],'r')

a: polynomial
  s^3 + 6 s^2 + 11 s + 6

   T = 1;
   b = c2d(a, T)

b: polynomial
  z^3 - 30.1929 z^2 + 223.0968 z - 403.4288

   roots(b)

ans =
    2.7183
    7.3891
   20.0855

   c = c2z(a, T)

c: polynomial
   z^3 - 0.553 z^2 + 0.074841 z - 0.0024788

   roots(c)

ans =
    0.3679
    0.1353
    0.0498
