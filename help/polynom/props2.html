Other properties of polynomials

Miscellaneous properties of polynomials can be obtained using the following functions.
Degree of a polynomial
Function deg returns the degree of a polynomial 
    p = poln ( [2 3 4], 'r' )
 
 p: polynomial
   s^3 - 9 s^2 + 26 s - 24
 
    dg = deg ( p )
 
 dg =
      3
For a quasipolynomial, this function returns maximal positive power and minimal negative power of the independent variable: 
    p = poln ( [2 3 4], 'rz', 1 )
 
 p: quasipolynomial
   z^2 - 9 z + 26 - 24 z^-1
 
    [dg,dm] = deg ( p )
 
 dg =
      2
 dm =
      1
Roots of a polynomial
Function roots returns the roots (zeros) of a polynomial or quasipolynomial: 
    p = poln ( [2 3 4], 'r' )
 
 p: polynomial
   s^3 - 9 s^2 + 26 s - 24
 
    r = roots ( p )
 
 r =
      2
      3
      4
Value of a polynomial
The overloaded function polyval returns the value of polynomial or quasipolynomial for a given value of the variable: 
    a = 2*s^2 + 3
 
 a: polynomial
   2 s^2 + 3
 
    v = polyval ( a, 2 )
 
 v =
     11
Continuous and discrete polynomials
Depending on the variable, polynomials can be considered as models continuous-time or discrete-time systems. Polynomials in the variables s and p are continuous, while polynomials in z, d, or q are called discrete.

Logical functions isct and isdt return 1 if the polynomial is continuous-time or discrete-time, respectively: 

    p = poln ( [1 2 3 )
 
 p: polynomial
   s^2 + 2 s + 3
 
    isct ( p )
 
 ans =
      1
 
    isdt ( p )
 
 ans =
      0
 
    p = poln ( [1 2 3, 'z' )
 
 p: polynomial
   z^2 + 2 z + 3
 
    isct ( p )
 
 ans =
      0
 
    isdt ( p )
 
 ans =
      1
