Creating polynomials

Using array of coefficients
Polynomials can be created using array of coefficients or roots. For example, the command 
    P = poln ( [2 -10 12] )
creates the polynomial 
P: polynomial
  2 s^2 - 10 s + 12
By default, the variable s is used. Other possible variables (s, p, z, d and q are allowed) can be specified as the second parameter of the constructor: 
    P = poln([2 -10 12], 'z')

P: polynomial
  2 z^2 - 10 z + 12
Using special variables
Special pre-defined variables s, p, z, d and q (realized as m-files) can also be used 
    P = 2*s^2 - 10*s + 12

P: polynomial
  2 s^2 - 10 s + 12
Using roots
Sometimes it is very important to specify roots of the polynomial instead of coefficients. Then, the second parameter should start with'r': 
    P = poln ( [2 3], 'r' )

P: polynomial
  s^2 - 5 s + 6

    P = poln ( [2 3], 'rz' )

P: polynomial
  z^2 - 5 z + 6
Roots of polynomials are stored as a data field of the object. If the roots are known exactly, such a representation makes it possible to enhance numerical properties of computation algorithms, for example, when cancelling a ratio of polynomials.

Creating Quasipolynomials

Using the same approach, it is possible to create quasipolynomials, i.e., rational functions having poles only at the origin. In this case, only variables z, d and q can be used. The third optional parameter of the constructor defines the so-called shift, i.e., the least negative degree with inverse sign. For example, 
    P = poln ( [2 -10 12], 'z', 2 )

P: quasipolynomial
  2 - 10 z^-1 + 12 z^-2
Special pre-defined variables s, p, z, d and q can be used: 
    Q = z + 1 + z^-1

Q: quasipolynomial
  z + 1 + z^-1
Quasipolynomials can also be defined via roots: 
    Q = poln([2 3], 'rz', 2)

Q: quasipolynomial
  1 - 5 z^-1 + 6 z^-2
