Factorization

Spectral factorization
In many optimal design procedures it is required to find a spectral factor of a Hermitian self-conjugate polynomial (or quasipolynomial) with respect to the imaginary axis or unit circle. Using the DirectSD Toolbox, this can be done with the following two functions: sfactor and sfactfft. The first of them is applicable to continuous-time polynomials 
   s = poln([1 0 2 0 9]);
   fs = sfactor ( s )

fs: polynomial
  s^2 + 2 s + 3
and symmetric quasipolynomials 
   s = 3*z^2 + 8*z + 14 + 8*z^-1 + 3*z^-2;
   fs = sfactor ( s )

fs: polynomial
  z^2 + 2 z + 3

   fs = sfactor ( s, 'z' )

fs: polynomial
  3 z^2 + 2 z + 1
If the second (optional) parameter is omitted or equals 'd', the function returns the factor having all poles outside the unit disk. If the second parameter is 'z', the function returns the factor having all poles inside the unit disk.

Function sfactfft implements a reliable method of factorizing discrete-time polynomials on the basis of FFT algorithm: 

   s = 3*z^2 + 8*z + 14 + 8*z^-1 + 3*z^-2;
   fs = sfactfft ( s )

fs: polynomial
  z^2 + 2 z + 3

   fs = sfactfft ( s, 'z' )

fs: polynomial
  3 z^2 + 2 z + 1
General factorization
General factorization is applicable to any polynomial (ot necessarily Hermitian self-conjugate), which can always be factorized onto stable, antistable and neutral cofactors. This can be done by means of the function factor for continuous-time polynomials 
   a = poln( [-1 2 0], 'r' )

a: polynomial
  s^3 - s^2 - 2 s

   [ps,pu,p0] = factor ( a )

ps: polynomial
  s + 1
pu: polynomial
  s - 2
p0: polynomial
  s
discrete-time polynomials: 
   a = poln( [0.3 1.5 1], 'rz' )

a: polynomial
  z^3 - 2.8 z^2 + 2.25 z - 0.45

   [ps,pu,p0] = factor ( a )

ps: polynomial
  z - 1.5
pu: polynomial
  z - 0.3
p0: polynomial
  z - 1
and quasipolynomials (shift is assigned to the antistable cofactor): 
   a = poln( [0.3 1.5 1], 'rz', 1 )

a: polynomial
  z^3 - 2.8 z^2 + 2.25 z - 0.45

   [ps,pu,p0] = factor ( a )

ps: polynomial
  z - 1.5
pu: quasipolynomial
  1 - 0.3 z^-1
p0: polynomial
  z - 1
