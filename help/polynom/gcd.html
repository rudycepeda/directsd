Common divisors

For many problems it is important to find common divisors of two or more polynomials. Function gcd returns the greatest common divisors of several polynomials: 
   a = poln ( [1 2 3], 'r' );
   b = poln ( [2 3 4], 'r' );
   c = poln ( [3 4 5], 'r' );
   g = gcd ( a, b, c )

g: polynomial
  s - 3
To extract this divisor from two or three polynomials, one can use functions coprime and triple, respectively. For the polynomials defined above: 
   [a1,b1,g] = coprime ( a, b )

a1: polynomial
  s - 1
b1: polynomial
  s - 4
g: polynomial
  s^2 - 5 s + 6
and also 
   [a2,b2,c2,g] = triple ( a, b, c )

a2: polynomial
  s^2 - 3 s + 2
b2: polynomial
  s^2 - 6 s + 8
c2: polynomial
  s^2 - 9 s + 20
g: polynomial
  s - 3
