Polynomial equations

Standard Diophantine equation
The standard scalar Diophantine polynomial equation has the form
where a, b, and c are known polynomials, while x and y are to be found. For solvability it is necessary and sufficient that the polynomial c be divisible by the greatest common divisor of a and b [1].

If the equation is solvable, there exist a family of solutions, which can be parametrized in terms of a free polynomial. Among them there are a solution with x of minimal degree, which can be found using the function dioph. Numerically, a linear polynomial equation is equivalent to a linear system of equations. For example, 

   a = s^2 + 3*s + 6;
   b = 2*s + 1;
   [x,y] = dioph(a, b, 1)
