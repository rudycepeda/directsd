Derivatives

Derivative of a polynomial
Function derive returns the derivative of a polynomial. The second (optional) parameter is the order of derivative: 
   a = 2*s^2 + 3*s + 4

a: polynomial
  2 s^2 + 3 s + 4

   derive ( a )

ans: polynomial
  4 s + 3

   derive ( a, 2 )

ans: polynomial
  4
This function is also applicable for quasipolynomials: 
   a = 2*z^2 + 3*z^-1 + 4*z^-2

a: quasipolynomial
  2 z^2 + 3 z^-1 + 4 z^-2

   derive ( a )

ans: quasipolynomial
  4 z - 3 z^-2 - 8 z^-3

   derive ( a, 2 )

ans: quasipolynomial
  4 + 6 z^-3 + 24 z^-4
Value of a derivative
The overloaded function polyder returns the value of a derivative of a polynomial or quasipolynomial for a given value of the variable: The third (optional) parameter is the order of derivative: 
   a = 2*s^3 + 3*s^2 + 4

a: polynomial
  2 s^3 + 3 s^2 + 4

   polyder ( a, 3 )

ans =
    72
Second-order derivative at the point s=3: 
   polyder ( a, 3, 2 )
   
ans =
    42
