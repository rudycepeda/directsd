DirectSD Toolbox    

ch2

Syntax

[K,cost] = ch2 ( sys )

[K,cost] = ch2 ( sys, method )

Description

ch2 returns the H2-optimal controller K for standard continuous-time system and the minimal value of the cost function cost. As distinct from h2reg, this function makes it possible to design controllers with imptoper transfer functions.

Arguments

sys model of an LTI plant associated with the standard continuous-time system.
method optional parameter that specifies solution algorithm. There are three possible methods:
  • 'ss'  - state-space solution [1];
  • 'wh'  - Wiener-Hopf method;
  • 'pol' - polynomial design method [2] (default).

Return values

K optimal continuous-time controller as an LTI model
cost minimal value of the cost function

Limitations

  • Controller K is a SISO system.

Example

   F = tf ( [1], [5 1 0] );
   Q = tf ( 1, [1 1] );
   R = tf ( 1, [1 0] );
   sys = [Q*R -F; R -F];
   [K,cost] = ch2 ( sys )

K: zero-pole-gain model 1 x 1

      5 (s+0.2)

cost =
     0

References

[2] Polyakov, K.Yu., Polynomial Design of Optimal Sampled-data Control Systems, Dr. Sc. Thesis, State University of Ocean Technology, St. Petersburg, 2006 [in Russian].
