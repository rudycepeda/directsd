polhinf

Syntax

[K,lam,Sigma] = polhinf ( A, B, E, D22 )

Description

polhinf solves H-problem
for a quadratic function in the frequency domain
where the function
is determined by the controller to be chosen.

Solution method is based on the discrete-time version of equalizing principle [1], [2], according to which the optimal X solves the H2-problem

for some constant λ>0 and function Σ(ζ) that is positive definite on the unit circle.

Arguments

A, B, E functions in the backward shift operator ζ.
D22 discrete-time model of the plant in the loop with hold curcuit

Return values

K discrete LTI model of controller
lam the minimal value of λ
Sigma the optimal function Σ(ζ)

References

[1]  Kwakernaak H., Minimax frequency domain performance and robustness optimisation of linear feedback systems // IEEE Trans. Automat. Contr., vol. AC-30, no. 10, pp. 994-1004, 1985.

[2] Grimble M.J., Robust Industrial Control: Optimal Design Approach for Polynomial Systems. UK: Prentice-Hall, Hemel Hempstead, 1994.

[3] Polyakov, K.Yu., Polynomial Design of Optimal Sampled-data Control Systems, Dr. Sc. Thesis, State University of Ocean Technology, St. Petersburg, 2006 [in Russian].
