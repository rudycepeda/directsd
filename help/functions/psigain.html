psigain

Syntax

[G,GK,a0,b0] = psigain ( sys, T )

[G,GK,a0,b0] = psigain ( sys, T, Delta )

Description

psigain returns the DC-gain of the parameter Ψ in the parameterization
(1)
such that the error ε can be made zero for t = . Here polynomials n(ζ) and d(ζ) are numerator and denominator of the discrete model of P22(s) with a zero-order hold H(s), while polynomials a0(ζ) and b0(ζ) satisfy the polynomial equation
(2)
where Δ(ζ) is a characteristic polynomial.

Arguments

sys model of an LTI plant associated with the standard sampled-data system.
T the sampling period
Delta characteristic polynomial of the closed-loop sampled-data system (default Δ(ζ)=1)

Return values

G DC-gain of the function Ψ in (1) (NaN if no restriction)
GK DC-gain of the controller (NaN if no restriction)
a0, b0 polynomials satisfying (2)

Limitations

  • Controller K is a SISO system.
  • Zero-order hold is used as an extrapolator.
  • Transfer matrices P11(s), P21(s), and P22(s) must be strictly proper, P12(s) must be at least proper.

Example

   Q = 1;
   R = tf(1, [1 0]);
   F = tf(1, [1 1 1]);
   sys = [-R*Q F;R*Q -F];
   T = 1;
   [G,GK] = psigain(sys, T)

G =
    0.3395
GK =
   Inf
