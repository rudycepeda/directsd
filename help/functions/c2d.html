c2d

Syntax

q = c2d ( p )

q = c2d ( p, T )

q = c2d ( p, T, var )

Description

c2d discretize a polynomial by changing its roots for their discretized version. A root ai is changed for zi=exp(-ai*T).

Arguments

pa polynomial
Tsampling period (default 1)
varvariable: 'z' (default), 'd' or 'q'

Return values

qdiscretized polynomial

Example

   a = poln([-1 -2 -3],'r')

a: polynomial
  s^3 + 6 s^2 + 11 s + 6

   b = c2d(a, 1)

b: polynomial
  z^3 - 30.1929 z^2 + 223.0968 z - 403.4288

   roots(b)

ans =
    2.7183
    7.3891
   20.0855
