coprime

Syntax

[a0,b0,g] = coprime ( a, b )

[a0,b0,g] = coprime ( a, b, tol )

Description

coprime extracts common factors of two polynomials.

Arguments

a, bgiven polynomials as arrays of coefficients or polynomial objects
toltolerance for cancellation (default 1e-8)

Return values

a0, b0 coprime polynomials
g the prime greatest common divisor such that a=a0g and b=b0g

Examples

Example 1. Initial data as arrays of coefficients. 
   a = poly([1 2 3]);
   b = poly([2 3 4]);
   [a0,b0,g] = coprime(a, b)

a0 =
    1.0000   -1.0000
b0 =
    1.0000   -4.0000
g =
    1.0000   -5.0000    6.0000
Example 2. Polynomial form of initial data. 
   a = poln([1 2 3], 'r');
   b = poln([2 3 4], 'r');
   [a0,b0,g] = coprime(a, b)

a0: polynomial
  s - 1
b0: polynomial
  s - 4
g: polynomial
  s^2 - 5 s + 6

References

[1] Kucera V., Discrete Linear Control, New York: Wiley, 1979.
