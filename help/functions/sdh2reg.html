sdh2reg

Syntax

[K,cost,p] = sdh2reg ( sys, T )

[K,cost,p] = sdh2reg ( sys, T, nmeas, ncon )

[K,cost,p] = sdh2reg ( sys, T, nmeas, ncon, options )

Description

sdh2reg constructs the H2-optimal digital controller K for standard sampled-data system using state-space techniques [1], [2].

Arguments

sys model of an LTI plant associated with the standard sampled-data system.
T the sampling period
nmeasnumber of measured signals (default 1)
nconnumber of control signals (default 1)
optionsstructure with the following fields:
  options.method - method of constructing equivalent:
  •  'gh2' - generalized H2-equivalent model (default)
  •  'sh2' - simple H2-equivalent model
  • 'fast' - model based on fast discretization

  options.Nfast - frequency multiplication factor for fast discretization (default 5)

Return values

K discrete LTI model of controller
costvalue of the cost function
proots of the closed-loop system characteristic polynomial in the 'z'-plane

Limitations

  • Only for delay-free systems.
  • Only for systems with zero-order hold.
  • Transfer matrices P11(s), P21(s), and P22(s) must be strictly proper, P12(s) must be at least proper.

Example

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    [K,cost,p] = sdh2reg(sys, T);
    K = zpk(K)
