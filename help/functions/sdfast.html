sdfast

Syntax

dsys = sdfast ( sys, T )

dsys = sdfast ( sys, T, nmeas, ncon )

dsys = sdfast ( sys, T, nmeas, ncon, Nfast )

dsys = sdfast ( sys, T, nmeas, ncon, Nfast, type )

Description

sdfast constructs a discrete-time equivalent model of the sampled-data system using fast discretization method.

Arguments

sys model of an LTI plant associated with the standard sampled-data system.
T the sampling period
nmeasnumber of measured signals (default 1)
nconnumber of control signals (default 1)
Nfastfrequency multiplication factor for fast discretization (default 5)
type discretization method:
  • 'std' - standard technique, discretize input and output [1] (default)
  •  'l2' - discretize input and preserve L2-norm of the output [2]

Return values

dsysgeneralized plant for an equivalent discrete-time standard system

Limitations

  • Only for delay-free systems.
  • Only for systems with zero-order hold.
  • Transfer matrices P11(s), P21(s), and P22(s) must be strictly proper, P12(s) must be at least proper.

Example

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    dsys = sdfast ( sys, T );
    K = h2reg ( dsys );
    K = zpk ( K )

K: zero-pole-gain model 1 x 1

      368.2916 (z-0.4561)
      -------------------
           (z+0.8743)

Sampling time: 0.1
