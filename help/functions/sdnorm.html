DirectSD Toolbox    

sdnorm

Syntax

val = sdnorm ( sys, K )

val = sdnorm ( sys, K, type )

Description

sdnorm returns a norm of sampled-data system with generalized plant sys and controller K.

Arguments

sys model of a continuous-time LTI plant associated with the standard sampled-data system.
K discrete-time feedback controller
type type of norm:
  •  'gh2' - generalized H2-norm (default)
  •  'sh2' - simple H2-norm
  •  'inf' - H-norm (L2-induced norm)
  • 'ainf' - associated H-norm

Return values

val value of the norm

Example

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk( 0.45, -0.89, 380, T);
    normH2 = sdnorm(sys, K)
    normH20 = sdnorm(sys, K, 'sh2')
    normHinf = sdnorm(sys, K, 'inf')
    normAHinf = sdnorm(sys, K, 'ainf')

normH2 =
    0.0287
normH20 =
    0.0375
normHinf =
    0.0092
normAHinf =
    0.0092
