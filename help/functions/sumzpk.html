sumzpk

Syntax

s = sumzpk ( f1, f2, ... )

Description

sumzpk constructs the sum of 'zpk'-models with common zeros and poles. It is more reliable than usual procedure.

Arguments

f1, f2, ... LTI models as objects of classes 'ss', 'tf' or 'zpk', the number of arguments is not limited

Return values

s sum of the functions in 'zpk'-form

Example

   f1 = tf([1 2 3], [1 3]);
   f2 = zpk([1 2], [-2 -3], 1);
   s = sumzpk ( f1, f2, 1 )

s: zero-pole-gain model 1 x 1

      (s+4.722) (s^2  + 1.278s + 2.965)
      ---------------------------------
                 (s+2) (s+3)
