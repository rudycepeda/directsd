DirectSD Toolbox    

sdsim

Syntax

[t,x,y] = sdsim ( sys, K, Tmax )

Description

sdsim simulate a linear standard sampled-data system and returns time, state and output vectors from sim function (Simulink).

Arguments

sys model of a continuous-time LTI plant associated with the standard sampled-data system.
K discrete-time feedback controller
Tmaxsimulation time

Return values

ttime vector
xstate vector
youtput vector

Example

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk( 0.45, -0.89, 380, T);
    Tmax = 1;
    [t,x,y] = sdsim ( sys, K, Tmax );
    plot(t, y)
