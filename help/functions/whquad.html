whquad

Syntax

[K,cost] = whquad ( A, B, E, D22 )

Description

whquad solves quadratic minimization problem
for a frequency domain integrand
where the function
is determined by the controller to be chosen.

Solution method is based on the Wiener-Hopf approach of [1].

Arguments

A, B, E functions in the backward shift operator ζ.
D22 discrete-time model of the plant in the loop with hold curcuit

Return values

K discrete LTI model of controller
cost the minimal value of the functional

Limitations

  • Controller K is a SISO system.
  • It is more reliable to use polquad or ssquad instead.

References

[1] Rosenwasser, E.N., and B.P. Lampe, Computer Controlled Systems: Analysis and Design with Process-orientated Models. London: Springer-Verlag, 2000.
