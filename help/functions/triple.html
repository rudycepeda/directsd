triple

Syntax

[a0,b0,c0,g] = triple ( a, b, c )

[a0,b0,c0,g] = triple ( a, b, c, tol )

Description

triple extracts common factors of three polynomials.

Arguments

a, b, cgiven polynomials as arrays of coefficients or polynomial objects
toltolerance (default 1e-8)

Return values

a0, b0, c0 triple of reduced polynomials
g the prime greatest common divisor such that a=a0g, b=b0g, and c=c0g

Examples

Example 1. Initial data as arrays of coefficients. 
   a = poly([1 2 3]);
   b = poly([2 3 4]);
   c = poly([3 4 5]);
   [a0,b0,g] = triple(a, b, c)

a0 =
    1.0000   -3.0000    2.0000
b0 =
    1.0000   -6.0000    8.0000
c0 =
    1.0000   -9.0000   20.0000
g =
    1.0000   -3.0000
Example 2. Polynomial form of initial data. 
   a = poln([1 2 3], 'r');
   b = poln([2 3 4], 'r');
   c = poln([3 4 5], 'r');
   [a0,b0,g] = triple(a, b, c)

a0: polynomial
  s^2 - 3 s + 2
b0: polynomial
  s^2 - 6 s + 8
c0: polynomial
  s^2 - 9 s + 20
g: polynomial
  s - 3

References

[1] Kucera V., Discrete Linear Control, New York: Wiley, 1979.
