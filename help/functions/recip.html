recip

Syntax

r = recip ( x )

Description

recip returns the reciprocal polynomial (the polynomial with inverse order of coefficients) such that for
the reciprocal polynomial is

Arguments

x a polynomial

Return values

rthe reciprocal polynomial

Example

   a = poln( [1 2 3 4], 'z' )

a: polynomial
  z^3 + 2 z^2 + 3 z + 4

   b = recip ( a )

b: polynomial
  4 z^3 + 3 z^2 + 2 z + 1
