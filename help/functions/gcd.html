gcd

Syntax

g = gcd ( p1, p2, ... )

Description

gcd returns a greatest common divisor of several polynomials.

Arguments

p1, p2, ...given polynomials as arrays of coefficients or polynomial objects

Return values

g the prime greatest common divisor of all polynomials

Examples

Example 1. Initial data as arrays of coefficients. 
   a = poly([1 2 3]);
   b = poly([2 3 4]);
   g = gcd(a, b)

g =
    1.0000   -5.0000    6.0000
Example 2. Polynomial form of initial data. 
   a = poln([1 2 3], 'r');
   b = poln([2 3 4], 'r');
   g = gcd(a, b)

g: polynomial
  s^2 - 5 s + 6

References

[1] Kucera V., Discrete Linear Control, New York: Wiley, 1979.
