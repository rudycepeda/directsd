vec

Syntax

v = vec ( p )

v = vec ( p, type )

Description

vec constructs the vector of coefficients for a polynomial of quasipolynomial.

Arguments

p polynomial as an array of coefficients or polynomial object.
type type of vectorization:
  • 'all' - all coefficients (default)
  •  '+'  - coefficients at non-negative powers (in ascending order)
  •  '-'  - coefficients at non-positive powers (in descending order)

Return values

v constant column-vector of coefficients

Examples

Example 1. Polynomial given as an array of coefficients. 
   p = [2 1 0];
   v = vec(p)

v =
     0
     1
     2
Example 2. Polynomial given as a polynomial object. 
   p = 2*s^2 + s;
   v = vec(p)

v =
     0
     1
     2
Example 3. Quasipolynomial as an object. 
   p = 2*z^2 + z - z^-1 -2*z^-2;
   v = vec(p)

v =
    -2
    -1
     0
     1
     2

   vp = vec(p, '+')

vp =
     0
     1
     2

   vm = vec(p, '-')

vm =
    -2
    -1
     0
