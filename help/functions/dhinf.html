dhinf

Syntax

[K,cost] = dhinf ( sys )

Description

dhinf returns the controller K that minimizes the H-norm of the standard discrete-time system, and the minimal value of the cost function cost.

Arguments

sys model of a discrete-time LTI plant associated with the standard discrete-time system.

Return values

K discrete LTI model of controller
cost minimal value of the cost function

Limitations

  • Controller K is a SISO system.

Example

    T = 1;
    F2 = tf ( 1, [-1 -2.1 1], T );
    F1 = tf ( [2 1 0], 1, T );
    Fw = tf ( [0.3 1], 1, T );
    V1 = tf ( 1, 1, T );
    V2 = tf ( 1, 1, T );
    sys = [V1*F2*Fw V1*F2*F1
               0       V2
            -F2*Fw   -F2*F1];
    sys = z2zeta(sys);
    [K,errOpt] = dhinf ( sys )

K: zero-pole-gain model 1 x 1

      1.6201 (z+0.4019)
      -----------------
          (z+1.302)

Sampling time: 1
