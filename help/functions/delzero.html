delzero

Syntax

[q,nz] = delzero ( p )

[q,nz] = delzero ( p, tol )

Description

delzero removes all zeros of a polynomial at the origin.

Arguments

pa polynomial
toltolerance (default eps)

Return values

qreduced polynomial
nznumber of removed zeros

Example

   a = poln( [1 2 0 0], 'z' )

a: polynomial
  z^3 + 2 z^2

   [b,nz] = delzero ( a )

b: polynomial
  z + 2
nz =
     2
