dtfm

Syntax

D = dtfm ( F, T, t )

D = dtfm ( F, T, t, type )

D = dtfm ( F, T, t, H )

D = dtfm ( F, T, t, type, H )

Description

dtfm returns the discrete Laplace transform for F(s) H(s), where F(s) is a proper function and H(s) is transfer function of a hold [1].

Arguments

F model of a continuous-time LTI plant with proper transfer function
Tsampling period
ttime parameter
H generalized hold given as an LTI continuous-time model.
type class of return value: 'tf', 'ss' or 'zpk' (default)

If the hold is not given, a zero-order hold is used by default.

Return values

D discrete Laplace transform with a hold curcuit

Example

    F = rss(2,1,1);
    T = 1;
    H = rss(1,1,1);
    H.d = 0;
    dtfm(F, T, 0, H)

References

[1] Rosenwasser, E.N., and B.P. Lampe, Computer Controlled Systems: Analysis and Design with Process-orientated Models. London: Springer-Verlag, 2000.
