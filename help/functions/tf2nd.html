tf2nd

Syntax

[n,d] = tf2nd ( f )

[n,d] = tf2nd ( f, var )

Description

tf2nd extracts numerator and denominator of a transfer function in the form of polynomial objects.

Arguments

f LTI model of a scalar plant (an object of classes 'ss', 'tf' or 'zpk')
var variable for polynomials: 's' (default for continuous-time systems), 'p', 'z' (default for discrete-time systems), 'd' or 'q'.

Return values

n, d numerator and denominator as polynomial objects

Examples

Example 1. State-space model: 
   f = rss(2,1,1);
   [n,d] = tf2nd ( f )
Example 2. Transfer function model: 
   f = tf([1 2 3], [4 5]);
   [n,d] = tf2nd ( f )

n: polynomial
  s^2 + 2 s + 3
d: polynomial
  4 s + 5
Example 3. Zero-pole-gain model. 
   T = 1;
   f = zpk([1 2 3], [4 5], 1, 1);
   [n,d] = tf2nd ( f, 'q' )

n: polynomial
  q^3 - 6 q^2 + 11 q - 6
d: polynomial
  q^2 - 9 q + 20
