bilintr

Syntax

b = bilintr ( f )

b = bilintr ( f, type )

b = bilintr ( f, type, T )

b = bilintr ( f, coef )

Description

bilintr performs bilinear transformation of a transfer function or zero-pole-gain model.

Arguments

f continuous-time or discrete-time LTI model
type type of transformation:
  • 's2z' - maps the left half-plane into the unit disk (default for continuous-time models)
  • 'z2s' - maps the unit disk into the left half-plane (default for discrete-time models)
  • 's2d' - maps the right half-plane into the unit disk (default for continuous-time models)
  • 'd2s' - maps the unit disk into the right half-plane (default for discrete-time models)
  • 'tustin' - direct and inverse Tustin transform
T sampling period for Tustin transform
coef array of four coefficients (a,b,c,d) for general transformation x = (ay+b)/(cy+d)

Return values

bbilinearly transformed function

Example

    F = tf ( 1, [1 1] );
    bilintr ( F )

ans: transfer function 1 x 1

      0.5 z + 0.5
      -----------
           z

Sampling time: 1

    bilintr ( F, 'tustin', 1 )

ans: transfer function 1 x 1

      0.3333 z + 0.3333
      -----------------
          z - 0.3333

Sampling time: 1

    bilintr ( F, [1 2 3 4] )

ans: transfer function 1 x 1

      0.75 z + 1
      ----------
        z + 1.5

Sampling time: 1
