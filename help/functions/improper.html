improper

Syntax

[p,f0] = improper ( f )

[p,f0] = improper ( f, type )

Description

improper extract improper part of a scalar transfer function.

Arguments

fscalar LTI model
typetype of separation:
  •   'p'  - f0 is proper but not necessarily strictly proper
  •  'sp'  - f0 is strictly proper (default)
  • 'symm' - f0 is strictly proper and Hermitian self-conjugate (like spectral density)

Return values

pimproper part (a polynomial)
f0proper or strictly proper part

Examples

Example 1. Extract improper and strictly proper terms. 
   f = tf([5 2 3],[1 2]);
   [p,f0] = improper ( f )

p: polynomial
  5 s - 8

f0: transfer function 1 x 1

        19
      -----
      s + 2
Example 2. Extract improper and proper terms. 
   f = tf([5 2 3],[1 2]);
   [p,f0] = improper ( f, 'p' )

p: polynomial
  5 s

f0: transfer function 1 x 1

      -8 s + 3
      --------
        s + 2
