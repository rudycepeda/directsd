dahinorm

Syntax

cost = dahinorm ( sys, K )

Description

dahinorm returns the associated H-norm for standard discrete-time system.

Arguments

sys model of an discrete-time plant associated with the standard system
K discrete-time feedback controller

Return values

cost AH-norm of the discrete-time system

Limitations

  • Controller K is a SISO system.

Example

    T = 1;
    G = tf ( [1 0],  [-2 1], T );
    V = tf ( [1 -2], [-2 1], T );
    W1 = tf ( 1, 1, T );
    W2 = tf ( 1, 1, T );
    sys = [W1*V W1*G
              0   W2
             -V   -G];
    sys = z2zeta(sys);
    K = zpk([], [], 1.5, T);
    cost = dahinorm ( sys, K )

cost =
    3.6056
