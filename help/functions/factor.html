DirectSD Toolbox    

factor

Syntax

[ps,pu,p0] = factor ( p )

[ps,pu,p0] = factor ( p, type )

Description

factor performs factorization of a (quasi)polynomial with respect to the imaginary axis or the unit circle. The function returns stable, antistable and neutral cofactors.

Arguments

p a (quasi)polynomial in the form of array of coefficients or polynomial object
type type of factorization:
  • 's' - with respect to the imaginary axis, stability region is the left half-plane (default for continuous-time models)
  • 'z' - with respect to the unit circle, stability region is inside the unit disk
  • 'd' - with respect to the unit circle, stability region is outside the unit disk (default for discrete-time models)

Return values

psstable factor
pumonic antistable factor
p0monic neutral factor

Examples

Example 1. Continuous-time polynomial. 
a = poln( [-1 2 0], 'r' )
a: polynomial
  s^3 - s^2 - 2 s
[ps,pu,p0] = factor ( a )
ps: polynomial
  s + 1
pu: polynomial
  s - 2
p0: polynomial
  s
Example 2. Discrete-time polynomial. 
a = poln( [0.3 1.5 1], 'rz' )
a: polynomial
  z^3 - 2.8 z^2 + 2.25 z - 0.45

[ps,pu,p0] = factor ( a )
ps: polynomial
  z - 1.5
pu: polynomial
  z - 0.3
p0: polynomial
  z - 1
Example 3. Discrete-time quasipolynomial (shift is assigned to the antistable cofactor). 
a = poln( [0.3 1.5 1], 'rz', 1 )
a: polynomial
  z^3 - 2.8 z^2 + 2.25 z - 0.45

[ps,pu,p0] = factor ( a )
ps: polynomial
  z - 1.5
pu: quasipolynomial
  1 - 0.3 z^-1
p0: polynomial
  z - 1
Example 4. Polynomial as an array. 
a = [1 -2.8 2.25 -0.45];
[ps,pu,p0] = factor ( a, 'z' )
ps =
    1.0000   -1.3000    0.3000
pu =
    1.0000   -1.5000
p0 =
     1
