sdl2err

Syntax

cost = sdl2err ( sys, K )

cost = sdl2err ( sys, K, H )

cost = sdl2err ( sys, K, method )

cost = sdl2err ( sys, K, H, method )

Description

sdl2err returns the L2-norm of the output for the standard sampled-data system:
It is assumed that the input w is a scalar Dirac delta-function δ(t).

Arguments

sys model of an LTI plant associated with the standard sampled-data system.
K feedback controller
H generalized hold given as an LTI continuous-time model.
method optional parameter that specifies solution algorithm. There are two possible methods:
  • 'ss'  - state-space solution (for delay-free systems only) [1];
  • 'pol' - polynomial design method [3] (default);

If the hold is not given, a zero-order hold is used by default.

Return values

cost L2-norm of the output ε

Limitations

  • Controller K is a SISO system.
  • Transfer matrices P11(s), P21(s), and P22(s) must be strictly proper, P12(s) must be at least proper.

Example

   F = tf ( 1, [1 -1] );
   R = tf ( 1, [1 0] );
   Q  = tf ( 1, [1 2] );
   T  = 0.5;
   sys = [-Q*R  F
             R -F];
   [K,cost] = sdl2 ( sys, T )
   cost = sdl2err ( sys, K )
