quaderr

Syntax

cost = quaderr ( A, B, E, M )

Description

quaderr computes the value of a quadratic cost function
for a frequency domain integrand
where the function
(1)
depends on the controller K(ζ). Here D22(ζ) is the discrete model of P22(s) with a hold (in the standard sampled-data system).

Arguments

A, B, E functions in the backward shift operator ζ.
M function (1) in the backward shift operator

Return values

K discrete LTI model of controller
cost the minimal value of the functional

References
