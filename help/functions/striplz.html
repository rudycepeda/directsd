striplz

Syntax

[b,n] = striplz ( a )

[b,n] = striplz ( a, tol )

Description

striplz removes leading zeros of a polynomial.

Arguments

a polynomial as an array of coefficients
tol tolerabce for removing zeros (default eps)

Return values

b polynomial without leading zeros
n number of removed leading zeros

Example

   a = [1e-10 0.1 1 20];
   [b,n] = striplz ( a )

b =
    0.0000    0.1000    1.0000   20.0000
n =
     0

   [b,n] = striplz ( a, 1e-8 )

b =
    0.1000    1.0000   20.0000
n =
     1
