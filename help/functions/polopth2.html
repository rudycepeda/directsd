DirectSD Toolbox    

polopth2

Syntax

[Q,E,Emin] = polopth2 ( V, W, n )

Description

polopth2 solves quadratic minimization problem
for stable rational functions V(ζ) and W(ζ). It is required to find the minimizing polynomial Q(ζ) of a known degree.

Solution method is based on the polynomial approach of [1] and [2].

Arguments

V, W stable functions in the backward shift operator ζ.
n degree of polynomial Q

Return values

Q optimal polynomial of degree n
E the minimal value of the H2-norm for fixed n
Emin the minimal value of the H2-norm for n = 

References

[1] Polyakov K., Rosenwasser E., and Lampe B., Two-level numerical procedure for optimal design of digital modal controllers // Proc. 44th IEEE Conf. Decision Contr. and ECC, Sevilla, Spain, pp. 4481-4486, 2005.

[2] Polyakov, K.Yu., Polynomial Design of Optimal Sampled-data Control Systems, Dr. Sc. Thesis, State University of Ocean Technology, St. Petersburg, 2006 [in Russian].
