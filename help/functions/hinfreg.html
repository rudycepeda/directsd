hinfreg

Syntax

[K,cost,CL] = hinfreg ( sys )

[K,cost,CL] = hinfreg ( sys, nmeas, ncon )

[K,cost,CL] = hinfreg ( sys, nmeas, ncon, options )

Description

hinfreg returns the controller K that minimizes the H-norm of the standard LTI system, and the minimal value of the cost function cost.

Arguments

sys model of a continuous-time or discrete-time LTI plant associated with the standard system.
nmeasnumber of measured signals (default 1)
nconnumber of control signals (default 1)
options additional parameters:
  • options.tol - tolerance (default 1e-4)
  • options.verbose - verbosity level (0-1, default 0)
  • options.method - solution method:
    • 'sa' - formulas by Safonov, Limebeer and Chiang (default)
    • 'gl' - formulas by Glover and Doyle

Return values

K LTI model of controller
cost minimal value of the cost function
CL model of the optimal closed-loop system

Example

    sys = drss ( 2, 2, 2);
    [K,cost,CL] = hinfreg ( sys )

References

[1] M.G. Safonov, D.J.N. Limebeer, and R.Y. Chiang, Simplifying H-theory via loop shifting, matrix pencil and descriptor concept, Int. J. Control, vol. 50, pp. 2467-2488, 1990.

[2] K. Glover, and J.C. Doyle, State-space formulae for all stabilizing controllers that satisfy an H-norm bound and relations to risk sensitivity, Systems and Control Letters, vol. 11, pp. 167-172, 1988.
