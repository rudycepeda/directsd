DirectSD Toolbox    

separss

Syntax

[fs,fu,ns,nu] = separss ( f )

[fs,fu,ns,nu] = separss ( f, type )

[fs,fu,ns,nu] = separss ( f, type, stype )

[fs,fu,ns,nu] = separss ( f, ps )

Description

separss performs proper separation of a rational function, extracting stable, antistable and neutral terms such that f=fs+fu.

Special syntax (the last line) is used when it is necessary to extract a strictly proper part with given poles.

The algorithm is based on state-space techniques.

Arguments

f scalar model of LTI system in 'ss', 'tf' or 'zpk' form
type type of separation:
  • 's' - with respect to the imaginary axis, stability region is the left half-plane (default for continuous-time models)
  • 'z' - with respect to the unit circle, stability region is inside the unit disk (default for discrete-time models)
  • 'd' - with respect to the unit circle, stability region is outside the unit disk
  • 's0', 'z0' and 'd0' - the same as 's', 'z' and 'd', but neutral poles are included in the stable term
stype defines where to place a polynomial term for non-strictly proper functions:
  • 'infs' - polynomial term is included in the stable part (default)
  • 'infu' - polynomial term is included in the antistable part
  • 'symm' - symmetric distribution between the stable and antistable parts
pspoles that should be placed in the first block

Return values

fsstable term (or term containing poles at ps)
fuantistable term (or term free of poles at ps)
nsdimension of state space for the first term
nudimension of state space for the second term

Examples

Example 1. Standard proper separaton. 
     F = zpk([3 4 -5 -3], [1 -1 -2 0], 1);
     [fs,fu] = separss ( F )

fs: zero-pole-gain model 1 x 1

      (s+65.76) (s+2.235)
      -------------------
          (s+1) (s+2)

fu: zero-pole-gain model 1 x 1

      -66 (s-1.364)
      -------------
         s (s-1)
Example 2. Separation with neutral poles. 
     F = zpk([3 4 -5 -3], [1 -1 -2 0], 1);
     [fs,fu] = separss ( F, 's0' )

fs: zero-pole-gain model 1 x 1

      (s-26.83) (s^2  + 4.834s + 6.708)
      ---------------------------------
                s (s+1) (s+2)


fu: zero-pole-gain model 1 x 1

        24
      -----
      (s-1)
Example 3. Special separation. 
     F = zpk([3 4 -5 -3], [1 -1 -2 0], 1);
     [fs,fu] = separss ( F, [1 -1] )

fs: zero-pole-gain model 1 x 1

      (s+104.5) (s-0.5452)
      --------------------
           (s-1) (s+1)

fu: zero-pole-gain model 1 x 1

      -105 (s+1.714)
      --------------
          s (s+2)
