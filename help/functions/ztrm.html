ztrm

Syntax

D = ztrm ( F, T, t )

D = ztrm ( F, T, t, type )

Description

ztrm returns the discrete Laplace transform D(z) for a strictly proper function F(s) [1]:
where T is the sampling period.

Arguments

F model of a continuous-time LTI plant with strictly proper transfer function
Tsampling period
ttime parameter
type class of return value: 'tf', 'ss' or 'zpk' (default)

Return values

D discrete Laplace transform of F

Example

    F = rss(2,1,1);
    T = 1;
    ztrm(F, T, 0)

References

[1] Rosenwasser, E.N., and B.P. Lampe, Computer Controlled Systems: Analysis and Design with Process-orientated Models. London: Springer-Verlag, 2000.
