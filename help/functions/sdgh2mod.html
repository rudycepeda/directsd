sdgh2mod

Syntax

[dsys,gamma] = sdgh2mod ( sys, T )

[dsys,gamma] = sdgh2mod ( sys, T, nmeas, ncon )

Description

sdgh2mod constructs a discrete-time equivalent model of the standard sampled-data system in generalized H2-problem [1], [2].

Arguments

sys model of an LTI plant associated with the standard sampled-data system.
T the sampling period
nmeasnumber of measured signals (default 1)
nconnumber of control signals (default 1)

Return values

dsysgeneralized plant for an equivalent discrete-time standard system
gammaadditional term in computation of H2-norm of sampled-data system

Limitations

  • Only for delay-free systems.
  • Only for systems with zero-order hold.
  • Transfer matrices P11(s), P21(s), and P22(s) must be strictly proper, P12(s) must be at least proper.

Example

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    [dsys,gamma] = sdgh2mod ( sys, T );
    K = zpk(h2reg ( dsys ))
    cost = sqrt(norm(lft(dsys,K))^2 + gamma)
