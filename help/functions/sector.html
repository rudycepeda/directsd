sector

Syntax

[alpha,beta] = sector ( p )

[alpha,beta] = sector ( p, dom )

[alpha,beta] = sector ( p, dom, T )

Description

sector computes degree of stability alpha and degree of oscillation beta (clipped sector in the s-plane) for given roots of the characteristic polynomial.

Arguments

p roots of the characteristic polynomial
dom domain:
  • 's' - continuous-time systems (default)
  • 'z' - discrete-time systems, forward shift operator
  • 'd' - discrete-time systems, backward shift operator
T sampling period for discrete-time systems (default 1)

Return values

alpha degree of stability for each element of p
beta degree of oscillation for each element of p

Example

   F = tf(1, [1 -1]);
   sys = [F F; -F -F];
   T = 1;
   rts = roots(charpol(sys, K))

rts =
   -0.2526

   [alpha,beta] = sector(rts, 'z', T)

alpha =
    1.3758
beta =
    2.2834
