sfactor

Syntax

fs = sfactor ( s )

fs = sfactor ( s, type )

Description

sfactor performs spectral factorization of a Hermitian self-conjugate (quasi)polynomial or rational function. Returns a stable factor, neutral terms are divided equally between stable and unstable cofactors.

Arguments

s Hermitian self-conjugate (quasi)polynomial or rational function
type type of factorization:
  • 's' - with respect to the imaginary axis, stability region is the left half-plane (default for continuous-time models)
  • 'z' - with respect to the unit circle, stability region is inside the unit disk
  • 'd' - with respect to the unit circle, stability region is outside the unit disk (default for discrete-time models)

Return values

fs stable spectral factor with stable inverse

Examples

Example 1. Continuous-time polynomial. 
   s = poln([1 0 2 0 9]);
   fs = sfactor ( s )

fs: polynomial
  s^2 + 2 s + 3
Example 2. Discrete-time quasipolynomial. 
   s = 3*z^2 + 8*z + 14 + 8*z^-1 + 3*z^-2;
   fs = sfactor ( s )

fs: polynomial
  z^2 + 2 z + 3

   fs = sfactor ( s, 'z' )

fs: polynomial
  3 z^2 + 2 z + 1
Example 3. Hermitian self-conjugate rational function. 
   s = tf( [-1 0 1], [1 0 -4 0 0]);
   fs = sfactor ( s )

fs: zero-pole-gain model 1 x 1
       (s+1)
      -------
      s (s+2)
