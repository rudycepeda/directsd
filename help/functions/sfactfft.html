sfactfft

Syntax

fs = sfactfft ( s, type )

fs = sfactfft ( s, type )

Description

sfactfft performs spectral factorization of a Hermitian self-conjugate quasipolynomial. Returns a stable factor.

Arguments

s Hermitian self-conjugate quasipolynomial or symmetric array of coefficients.
type type of factorization:
  • 'z' - stability region is inside the unit disk
  • 'd' - stability region is outside the unit disk (default)

Return values

fs stable polynomial spectral factor

Examples

Example 1. Quasipolynomial as an array of coefficients. 
   s = [3 8 14 8 3]
   fs = sfactfft ( s )

fs =
    1.0000    2.0000    3.0000
Example 2. Quasipolynomial as an object . 
   s = 3*z^2 + 8*z + 14 + 8*z^-1 + 3*z^-2;
   fs = sfactfft ( s )

fs: polynomial
  z^2 + 2 z + 3

   fs = sfactfft ( s, 'z' )

fs: polynomial
  3 z^2 + 2 z + 1

References
