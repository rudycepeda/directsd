DirectSD Toolbox    

deg

Syntax

d = deg ( a )

Description

deg returns degree of a polynomial or order of a scalar transfer function. In the latter case the order is defined as the maximum of degrees of the numerator and denominator.

Arguments

aa polynomial (as an array of coefficients or polynomial object) or scalar LTI model

Return values

d degree of a polynomial or order of a transfer function

Examples

Example 1. Degree of a polynomial given as an array of coefficients. 
   a = [1 2 3];
   d = deg(a)

d =
     2
Example 2. Degree of a polynomial given as an object. 
   a = s^2 + 3*s + 3;
   d = deg(a)

d =
     2
Example 3. Order of a scalar transfer function. 
   a = rss(2, 1, 1);
   d = deg(a)

d =
     2
