dinfnorm

Syntax

[cost,w] = dinfnorm ( sys )

[cost,w] = dinfnorm ( sys, tol )

Description

dinfnorm returns the H-norm of a discrete-time system.

Arguments

sys model of a discrete-time LTI plant.
tol tolerance (default 1e-3)

Return values

cost H-norm of the discrete-time system
w frequency at which the maximum is reached

Example

    sys = drss ( 2, 1, 1 );
    [cost,w] = dinfnorm( sys )

References

[1] Bruisma, N.A., and M. Steinbuch, A fast algorithm to compute the H-norm of a transfer function matrix, Syst. Contr. Letters, vol. 14, pp. 287-293, 1990.
