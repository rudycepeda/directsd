H∞ norm

The following L2-induced norm
is usually meant by the H-norm of a sampled-data system. This equals the supremum of L2-norm of the output for all inputs with bounded L2-norm.

For sampled-data systems with time delays computation of the H-norm and solution of the corresponding optimization problem can be problematic. As an alternative, the associated H-norm can be used.

Computation

Function sdhinorm returns the H-norm of the standard sampled-data system with continuous-time plant sys and discrete-time controller K: 
    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk ( 0.45, -0.89, 380, T );
    v = sdhinorm(sys, K)

v =
    0.0092
