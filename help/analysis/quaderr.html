Integral quadratic error

Integral quadratic error
for a given input signal is a widely used performance criterion.

Without loss of generality, it can be assumed that signal generator is incorporated into the augmented plant of the standard sampled-data system so that the input w is a scalar impulse δ(t) (Dirac delta-function). In this case, it can be proved that the integral quadratic error J coincides with the square of the simple H2-norm. Therefore, this value can be computed using the function sdnorm with optional parameter 'sh2': 

   F = tf(1, [1 1 0]);
   sys = [F F;-F -F];
   T = 0.1;
   K = zpk ( 0.45, -0.89, 380, T );
   v = sdnorm(sys, K, 'sh2')^2

v =
    0.0014
or by means of the function sdl2err: 
   v = sdl2err(sys, K)

v =
    0.0014
