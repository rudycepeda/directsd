H2-norm

For LTI systems, one of the most popular is the problem of minimizing the H2-norm of closed-loop transfer function [1]. The notion of transfer function is senseless for sampled-data systems, because they are periodically nonstationary systems. A family of values
can be considered, where where εiτ denotes system response to a delayed Dirac delta-function δ(t-τ) applied to the ith input (i = 1,...,m). For LTI systems this value is independent of τ, but for sampled-data systems this is a periodic function of τ. Therefore, the following generalized H2-measure (or H2-norm) can be used
which characterize system behaviour in continuous-time [2]-[4].

It can be proved that this value is equal to average variance of stochastic error signal under centered unit white noise at the input.

Computation

Function sdnorm with optional parameter 'gh2' returns the H2-norm of the standard sampled-data system with continuous-time plant sys and discrete-time controller K: 
    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk ( 0.45, -0.89, 380, T );
    v = sdnorm(sys, K, 'gh2')

v =
    0.0287
For a system with scalar controller this value can be computed in another way by means of the function sdh2norm: 
   v = sdh2norm(sys, K)

v =
    0.0287
