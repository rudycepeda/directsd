For sampled-data systems it is known [2] that H2 and L2 problems can also be reduced, in the frequency domain, to a functional of the form (1), where X(ζ) is a Hermitian self-conjugate function free of poles at the unit circle. Therefore, a stable function F(ζ) can be found such that (3) holds. This function can be considered as transfer function of an equivalent discrete-time system, and H-norm of this function defined by (2) can be taken as a measure of robustness for the sampled-data system under consideration. Advantages

  • AH norm can be computed effectively in the frequency domain for a wide class of sampled-data systems, including time-delay systems.
  • AH norm coincides with the classical L2-induced norm [3] for systems with P11(s)=0.

Disadvantages

  • AH norm is different from the classical L2-induced norm [3] for systems P11(s)≠0.

Computation

Function sdahinorm returns the AH-norm of the standard sampled-data system with continuous-time plant sys and discrete-time controller K: 
    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk ( 0.45, -0.89, 380, T );
    v = sdahinorm(sys, K)

v =
    0.0092
