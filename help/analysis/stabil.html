Stability

Characteristic polynomial

Stability of sampled-data system is determined by characteristic polynomial on the basis of discrete-time model of the block P22 with hold H in the standard sampled-data system.

Function charpol returns the characteristic polynomial in the z-plane of the standard sampled-data system with continuous-time plant sys and discrete-time controller K: 

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk ( 0.45, -0.89, 380, T );
    p = charpol(sys, K)

p: polynomial
  z^3 + 0.82338 z^2 + 0.16029 z + 0.0052236
For stability of the closed loop it is necessary and sufficient that all roots of the characteristic polynomial be located inside the unit circle. For the above example we find: 
   rts = roots ( p )

rts =
   -0.5485
   -0.2342
   -0.0407
so that the system is stable.

Stability margin

To find the stability margin, it is necessary to extract the uncertainty block as shown below:
Then, the stability margin is defined as the inverse of the H-norm of the standard sampled-data system (without the Δ-block). This value equals the maximal admissible L2-induced norm of the periodically nonstationary perturbations described by the Δ-block, under which the closed-loop system remains stable.

Function sdmargin returns the stability margin for the standard sampled-data system with continuous-time plant sys and discrete-time controller K: 

    F = tf(1, [1 1 0]);
    sys = [F F;-F -F];
    T = 0.1;
    K = zpk ( 0.45, -0.89, 380, T );
    g = sdmargin (sys, K)

g =
  108.8637
