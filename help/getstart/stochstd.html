Stochastic problem

The augmented continuous-time plant of the standard system must incorporate all shaping filters (generators of external signals) and weighting functions. For stochastic problems, the input signal is a vector white noise with identity spectral density matrix. This causes no loss of generality, is the the input signal has a rational spectral density matrix. Then, the corresponding shaping filter must be included in the augmented plant.

    sys = [ F*Fw  F
             0    rho
