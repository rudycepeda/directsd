Deterministic problem

The augmented continuous-time plant of the standard system must incorporate all shaping filters (generators of external signals) and weighting functions. For deterministic problems, the input signal is a scalar unit impulse (Dirac delta-function). This causes no loss of generality, if the input signal can be described by a Laplace image. Then, the corresponding filter must be included in the augmented plant.

Consider the system with
and a zero-order hold. Let the sampling period be T = 1 and ρ = 0.2. These data can be entered as follows: 
   F1 = tf ( 1, [25 1] );
   F2 = tf ( 1, [1 0] );
   R = tf ( 1, [1 0] );
   rho = 0.2;
   T  = 1;
The plant for the standard system is 
   sys = [R  -F2*F1
          0    rho
          R  -F2*F1];
