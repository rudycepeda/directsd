Limitations

  • Signals y and u are assumed to be scalar, so that controller K is scalar as well.
  • Transfer matrices P11(s), P21(s), and P22(s) must be strictly proper, P12(s) must be at least proper.

By default, it is assumed that the hold device H is a zero-order hold.

Characteristic polynomial for this system can be found using charpol: 

   cp = charpol ( sys, K )

cp: polynomial
  z^3 - 2.1411 z^2 + 1.3626 z - 0.2019
It can be easily found that all its roots are inside the unit disk so that the system is stable: 
    roots(cp)

ans =
   0.2115
   0.9648 + 0.1537i
   0.9648 - 0.1537i
The H2-norm of the system can be found using sdh2norm: 
   sdh2norm ( sys, K )

ans =
    0.8332
Using sdh2 one can find the optimal controller minimizing the H2-norm: 
   Kopt = sdh2 ( sys, T )

Kopt: zero-pole-gain model 1 x 1

      105.4099 (z-0.3867) (z-0.1501)
      ------------------------------
           (z+0.9186) (z-0.1347)

Sampling time: 1
For the optimal K we have 
    sdh2norm ( sys, Kopt )

ans =
    0.0292
i.e., the H2-norm was decreased more than 28 times.
