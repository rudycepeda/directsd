L2-optimization problem

Integral quadratic error
for a given input signal is a widely used performance criterion [1], [2]. This equals the L2-norm of the error signal, therefore, the problem of minimizing this value will be called the L2-optimization problem for sampled-data systems.

Without loss of generality, it can be assumed that signal generator is incorporated into the augmented plant of the standard sampled-data system so that the input w is a scalar impulse δ(t) (Dirac delta-function). In this case, it can be proved that the integral quadratic error J coincides with the square of the simple H2-norm.

Polynomial method of solving the L2-problem for sampled-data systems [3] is realized in the function sdl2, which returns the optimal digital controller and the minimal attainable value of the cost function: 

   F = tf(1, [1 1 0]);
   sys = [F F;-F -F];
   T = 0.1;
   [K,cost] = sdl2 ( sys, T )
