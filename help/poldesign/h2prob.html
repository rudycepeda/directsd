H2-optimization problem

Fro the engineering point of view, the H2-problem for sampled-data systems is more understandable in stochastic setting. The design goal is to minimize the value
where E{·} denotes mathematical expectation and the input signal w is described as centered white noise with unit spectral density [1]. In deterministic setting this equals the H2-norm of the system [2].

Polynomial method of solving the H2-problem for sampled-data systems [3] is realized in the function sdh2, which returns the optimal digital controller and the minimal attainable value of the cost function: 

   F = tf(1, [1 1 0]);
   sys = [F F;-F -F];
   T = 0.1;
   [K,cost] = sdl2 ( sys, T )
