Associated H∞ problem

Associated H-problem consists in minimizing the AH-norm of sampled-data system [1], [2]:
where U denotes the causal periodically nonstationary operator corresponding to the sampled-data system, and F(ζ) denotes transfer function of an equivalent discrete-time system.

AH norm can be computed effectively in the frequency domain for a wide class of sampled-data systems, including time-delay systems.

AH norm coincides (up to a constant factor) with the classical H (L2-induced) norm [3] only for systems with P11(s) = 0. Nevertheless, computations show that in many cases optimization on the basis of minimizing H and AH-norms give quite similar results.

Polynomial method of solving the AH-problem for sampled-data systems [4] is realized in the function sdahinf, which returns the optimal digital controller and the minimal attainable value of the cost function: 

   F = tf(1, [1 1 0]);
   sys = [F F;-F -F];
   T = 0.1;
   [K,cost] = sdahinf ( sys, T )
