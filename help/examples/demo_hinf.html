H2-optimal controller

Choose a sampling period: 
   T = 1;
To minimize the H2-norm of the system (this is equivalent to minimizing the chosen cost function), we use sdh2: 
   K2 = sdh2 ( sys, T )


K2: zero-pole-gain model 1 x 1

      -----------------
          (z-0.8048)

Sampling time: 1
For the system with this controller, we compute the H2-norm and AH-norm, using the functions sdh2norm and sdahinorm, respectively: 
   sdh2norm(sys, K2)

ans =
    3.2563

   sdahinorm(sys, K2)

ans =
   11.4122
For comparison, we compute also the H (or L2-induced norm) sdhinorm, 
   sdhinorm ( sys, K2 )

ans =
   11.4082
which is very close to the AH-norm.

AH-optimal controller

Then we solve the associated AH-problem for the stochastic problem considered above. Using sdahinf yields: 
   Kinf = sdahinf(sys, T)
