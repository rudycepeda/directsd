H2-optimization

Now we compute the controller that minimizes J so that the second requirement is ignored. Choose a sampling period: 
   T = 1;
To minimize the H2-norm of the system (this is equivalent to minimizing the chosen cost function), we use sdh2: 
   KH2 = sdh2 ( sys, T )

Mixed H2/ AH-optimization

To provide for a trade-off between the two criteria, we use the function sdh2hinf. The weighting coefficient ρ ∈ [0,1] makes it possible to provide for a trade-off between stochastic and robust criteria. For ρ = 0 we obtain the AH-optimal controller, while for ρ = 1 the H2-norm is minimized.

Choosing ρ = 0.5, we find: 

   Kmix = sdh2hinf ( sys, T, 0.5, 2, 1 )

Kmix: zero-pole-gain model 1 x 1

      5.8253 (z-0.7217) (z^2  - 1.402z + 0.5314)
      ------------------------------------------
          (z+0.5234) (z^2  - 1.427z + 0.547)

Sampling time: 1
The H2-norm and AH-norm for this system are computed as 
   sdh2norm ( sys, Kmix )

ans =
    0.9498

   sdahinorm ( sys, Kmix )

ans =
    1.3436
