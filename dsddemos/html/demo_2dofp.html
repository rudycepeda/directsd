Preparing data

First, we enter the models of the command signal, plant and reference system: 
   R = tf ( 1, [1 1 0] );
   F = tf ( 1, [5 1] );
   F.iodelay = 1.5;
   Q = tf ( 1, [0.1 1] );
   T = 1;
Let the preview interval π be equal to 2: 
   preview = 2;
Then, we represent the preview in the form
where σ is a non-negative integer and 0 ≤ θ  T. It can be shown that non-causal preview block can be removed and the equal delay be added to the ideal operator Q. In addition, a pure delay by θ should be added to the reference block (see [1]-[2] for details). Therefore, 
   sigma = ceil(preview/T);
   theta = sigma*T - preview;
   R.iodelay = theta;
   Q.iodelay = preview;

2-DOF controller

Then, we use 2-DOF control, assuming that the feedback controller K is fixed as defined above. Using sd2dof, we find the optimal reference controller: 
   KR = sd2dof ( sys2, K )

KR: zero-pole-gain model 1 x 1

      6.2878 z (z+6.675e-006) (z-9.713e-005) (z-0.0896) (z-0.8187)
      ------------------------------------------------------------
         (z-1) (z-0.2725) (z-4.54e-005) (z^2  + 1.943z + 1.248)

Sampling time: 1
Compute the cost function using sd2doferr 
   sd2doferr ( sys2, K, KR )

    0.0602
